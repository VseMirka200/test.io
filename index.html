Основные возможности

Конвертация документов

Конвертируйте документы между форматами: DOC/DOCX ↔ PDF, PDF ↔ ODT, Word ↔ ODT, PDF в изображения и обратно.

Сжатие файлов

Оптимизируйте изображения и PDF файлы с настраиваемым уровнем сжатия. Поддержка Ghostscript и PyMuPDF.

Переименование файлов

Пакетное переименование с использованием шаблонов: добавление префиксов/суффиксов, нумерация, замена текста.

Drag-and-Drop

Просто перетащите файлы в окно программы для начала работы. Интуитивно понятный интерфейс.

Экспорт и импорт

Экспортируйте и импортируйте шаблоны переименования. Работа с пользовательскими настройками.

Интеграция с Windows

Добавление в контекстное меню Windows. Быстрый доступ к программе из Проводника.

Системные требования

Основные требования

  • Windows 7/8/10/11 или Linux/Mac
  • 1 ГБ оперативной памяти
  • 100 МБ свободного места на диске
  • Python 3.8+ (для версии скрипта)

Для конвертации PDF

  • Ghostscript (рекомендуется)
  • LibreOffice (опционально)
  • PyMuPDF библиотека

Для работы с изображениями

  • Pillow (PIL) библиотека
  • pdf2image (для PDF→Изображения)
  • Доступ к системным шрифтам

Дополнительные библиотеки

  • docx2pdf (Word→PDF)
  • pdf2docx (PDF→Word)
  • python-odf (работа с ODT)
  • comtypes (конвертация Office)
